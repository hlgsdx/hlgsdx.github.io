可以使用间接引用的方法使用它：“* 指针变量名”

指针的本质

指针的本质是一个存地址的变量，在32位程序中，大小为8个16进制数（4字节）；在64位程序中，大小为16个16进制数（8字节），与指向的类型无关。

p 等价于 &num

*p 等价于 *&num 等价于 num

分析指针变量的类型方法

变量的类型：去掉变量名

指针指向的类型：去掉变量名 再去掉一个*，剩下的就是指针指向的类型。

int *p 变量的类型 int* ；指针指向的类型 int。

int **pp 变量的类型 int ** ；指针指向的类型 int *。

野指针

野指针是指向一个已删除的对象或未申请访问受限内存区域的指针。与空指针不同，野指针无法通过简单地判断是否为 NULL避免。所以，对野指针进行操作很容易造成程序错误。

写程序时要尽量避免操作自己的程序以外的地址。
