算法题 数组中唯一出现1次的数|唯一出现2次的数

参考：https://www.cnblogs.com/wangzaizhen/p/5161762.html
有一数组，仅有一个元素出现1次，其他元素都出现了偶数次。问如何快速找到该元素？

利用异或的特性，x^y^x=y^x^x=y。对数组所有元素一次进行异或操作，最终得到的值就是那个只出现一次的数字

若是有两个元素只出现1次呢？

假设两个只出现一次的数字分别为a、b，对数组元素进行两两异或后得到的值为c,可知c=a^b。因为a和b一定不同，所以c不等于0，那么c的二进制表示中一定有一位是1，即a,b在此位上分别为0,1或1,0。假设该位置是从低位起第x位，那么将数组分成两组，一组中的数字在第x位是0，另一组中的数字在第x位是1，则a和b分别属于两个子数组。分别求子数组中唯一只出现1次的数字，即可得到a和b

数组元素的访问方法

数组名 + 下标 数组名[下标数字]
注：下标从0开始，即若有10个元素，下标0~9

一般情况下 数组的名字是首元素的地址
当数组名与&和sizeof()结合的时候，代表整个数组

数组[]访问原理

数组名+下标 ===== 偏移量 + 间接引用
[]的作用 ： 先偏移量 然后间接引用

arr[0] ==== (arr + 0)
arr[1] ==== (arr + 1)

特殊情况
当间接引用整个数组的地址 会拿到首元素的地址

分析：

先取整个数组的首地址，偏移一个该数组大小，到达后一个相同数组大小首地址，间接引用得到首元素地址，再向前偏移一个元素大小，再间接引用，拿到9。
