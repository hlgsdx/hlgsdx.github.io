字符串

以’\0’结尾的字符数组；
\0 、 0 和 ‘0’区别
\0： ASCII 为 0 的字符，是 字符串结束符，是转义字符。
0：ASCII码为0，字符表示为空字符，NULL；数值表示为0
‘0’：字符0，它的ASCII码是48

练习 编写函数输出字符串
参数：字符数组首地址
返回值：成功输出的字符数；失败返回-1

程序内存空间分配

  • 栈区
  • 堆区
  • 全局/静态区
    数据段
    BSS
  • 字符常量区

栈区：
声明的局部变量
生命周期：定义变量的时候出生（开辟内存空间） 所在的作用域结束时消亡（系统回收）

堆区：
动态申请空间 申请者自己手动释放
malloc()申请空间 （连续空间） 返回首地址void* 泛型指针
free()释放 只能释放一次 可以free(NULL)
生命周期 malloc()出生 free()消亡
